# Commit Types and Emoji Reference

## Standard Types

| Type | Emoji | Description | Use When |
|------|-------|-------------|----------|
| `feat` | ✨ | New feature | Adding user-facing functionality |
| `fix` | 🐛 | Bug fix | Fixing a bug |
| `docs` | 📝 | Documentation | Changes to documentation |
| `style` | 💄 | Formatting/style | Code formatting, missing semicolons, etc. |
| `refactor` | ♻️ | Code refactoring | Restructuring code without changing behavior |
| `perf` | ⚡️ | Performance | Performance improvements |
| `test` | ✅ | Tests | Adding or updating tests |
| `chore` | 🔧 | Tooling, configuration | Build process, dependencies, tooling |
| `ci` | 🚀 | CI/CD | CI/CD configuration and scripts |
| `revert` | ⏪️ | Revert changes | Reverting a previous commit |
