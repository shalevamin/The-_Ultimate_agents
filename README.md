## ūüöÄ What Does the Installer Do?

The one-line installer runs a **7-phase onboarding process**:

| Phase               | What Happens                                                                                                          |
| ------------------- | --------------------------------------------------------------------------------------------------------------------- |
| **1. Consent**      | ‚ö†ÔłŹ Displays danger warning ‚ÄĒ you must type `I UNDERSTAND` to proceed                                                  |
| **2. System Scan**  | ūüĒć Checks what's already installed (Homebrew, Node, Python, etc.) and skips what exists                               |
| **3. Permissions**  | ūüĒź Pops up macOS dialogs for Accessibility, Screen Recording, and Admin access                                        |
| **4. API Key**      | ūüĒĎ Prompts for your OpenAI API key + lets you pick a model (gpt-4o, o1, etc.)                                         |
| **5. Install**      | ūüď¶ Installs only the missing dependencies (Homebrew, Node, Python, Playwright, Codex CLI)                             |
| **6. Verification** | ‚úÖ Runs checks and shows a pass/fail summary                                                                          |
| **7. Launch Menu**  | ūüöÄ Presents a dashboard with options to start the agent, connect WhatsApp/Telegram/Discord/Slack, or launch Codex CLI |

## ‚öôÔłŹ Configuration

### Environment Variables (`.env`)

| Variable            | Required | Description                                 |
| ------------------- | -------- | ------------------------------------------- |
| `OPENAI_API_KEY`    | ‚úÖ Yes   | Your OpenAI API key (the only required key) |
| `TUA_MODEL`         | No       | Default AI model (set during install)       |
| `ANTHROPIC_API_KEY` | No       | For Claude-based features                   |
| `GOOGLE_API_KEY`    | No       | For Google Imagen/Gemini images             |

### Model Selection

Choose during installation or edit `.env`:

| Model         | Best For                            |
| ------------- | ----------------------------------- |
| `gpt-4o`      | Best quality + vision (recommended) |
| `gpt-4o-mini` | Faster, cheaper                     |
| `o1`          | Complex reasoning                   |
| `o1-mini`     | Fast reasoning                      |
| `gpt-4.1`     | Latest flagship                     |
